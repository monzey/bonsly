# Monzey's dotfiles

Currently supports the following confs:
+ alacritty
+ dunst
+ feh
+ i3
+ ncmpcpp
+ polybar
+ qutebrowser
+ ranger
+ rofi
+ st
+ tig
+ tmux
+ vim
+ x
+ zsh

## From a fresh install of debian 11 bullseye

```bash
  apt install git
  git clone https://github.com/monzey/dotfiles.git
  cd dotfiles
  chmod a+x init.sh
  ./init.sh
```

That's it !
